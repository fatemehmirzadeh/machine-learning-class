`S{3+,3-} so Entropy(S)=1`

`a1 (true {2+,1-} , false {1+,2-})`

`Entropy (a1=true)=-{2/3log(2/3)+1/3log(1/3)}=0.9183`

`Entropy (a1=false)= 0.9183`

`gain(S,a1)=1-3/6*0.9183-3/6*0.9183=0.0817`

`a2 (true {2+,2-} , false {1+,1-})`

`Entropy (a2=true)=1`

`Entropy (a2=false)=1`

`gain(S,a2)=1-/6*0-2/6*0=0`
